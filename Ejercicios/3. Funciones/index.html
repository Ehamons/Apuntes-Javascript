Ejercicios usando funciones

Para mirar los ejecicios por favor abra la consola

Ejercicio: Dueño y mascota

Crear una funcion EXPRESADA que devuelva el nombre de ustedes, el nombre de sus mascotas y la edad de nosotros

Ejercicio: Calculadora básica

haga una función expresada para cada una de las siguientes operaciones

  1. Suma
  2. Resta
  3. Multiplicación
  4. División

seleccionar la operación que se quiere realizar haga un menú con la estructura de control

Ejercicio :Comparador de tamaño de números

Haga un código que con funciones expresadas haga una función relacion(a, b) que a partir de dos números cumpla lo siguiente:

  1. Si el primer número es mayor que el segundo, debe devolver 1.
  2. Si el primer número es menor que el segundo, debe devolver -1.
  3. Si ambos números son iguales, debe devolver un 0.
