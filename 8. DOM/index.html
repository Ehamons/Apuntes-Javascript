DOM.

Este modelo de trabajo permite modificar todos los objetos que tenga en algun documento HTML por medio de los selectores.

Selectores

  • Clases (class): Tienen por caracteristica que pueden asignarse a múltiples elementos
  • IDs (id): Tienen la característica de que son únicos, es decir, sólo se les puede asignar un elemento
  • Etiquetas (h1, p, img, div, section, ul, li, ...)
